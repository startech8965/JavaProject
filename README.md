## Installation

### 1. Clone the Repository
```bash
git clone https://github.com/startech8965/JavaProject.git
cd JavaProject
```

### 2. Build the Project
Using Maven:
```bash

```

Or using Gradle:
```bash

```

### 3. Run the Application
